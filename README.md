# Сугако Татьяна 253503
### Описание: 
Аналог Todoist. Приложение для мониторинга задач с уведомлениями и удобной системой классификации.

### Авторизация - вход пользователя в систему
   - register() - регистрация нового пользователя
   - login() - регистрация пользователя
   - createUser() - создание экземпляра класса User на текущую сессию

### User - пользователь
информация о пользователе:
  - picture - фото профиля
  - name - имя
  - password - пароль
  - dailyGoal - цель на день (отдельный класс)
  - tasks - список задач (отдельный класс)
  - projects - список пользовательских проектов (отдельный класс) методы:
  - edit() - изменить информацию о пользователе
  - logout() - выйти из системы
  - addTask() - добавление задачи
  - editTask() - изменить задачу
  - deleteTask() - удалить задачу
  - completeTask() - выполнить задачу
  - addProject() - добавление проекта
  - editProject() - изменить проект
  - deleteProject() - удалить проект

### Daily Goal - цель задач на день
  - dailyGoal - число задач, которое необходимо выполнить для достижения цели
  - changeDailyGoal() - изменить цель на день

### Task - задача пользователя
поля:
  - name - название задачи
  - description - описание
  - deadline - срок выполнения (бессрочный, конкретная дата и время, есть возможность задать повторяющуюся задачу)
  - notification - напоминания (одно или несколько)
  - label - метка (кастомная категория задач)
  - project - проект (по умолчанию - Входящие)
  - completed - статус задачи (выполнена да нет)
    
### Notification - уведомления
   - time - время
   - display() - показать уведомление

### Label - метка
   - color - цвет
   - name - имя

### Display - отображение
  - position - расположение задач в разных видах (список или доска)
  - sorting - сортировка задач по различным параметрам (вручную, название, срок выполнения)
  - filtering - фильтрация - отображение тех задач, что соответствуют параметрам (метка, приоритет)

### Position
   - positionList() - отобразить в виде списка
   - positionBoard() - отобразить в виде доски

### Sortiing
   - sortByName() - сортировать по названию задач
   - sortByDeadLine() - сортировать по сроку выполнения

### Filtering
   - filterByLabel() - показать только определенные метки
   - filterByPriority() - показать по приоритету
